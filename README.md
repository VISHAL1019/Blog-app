# Full Stack Blog Application

A full-stack blog platform built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

## Features
- User Signup/Login
- JWT Authentication
- Create Blog Posts
- View Blog Posts
- Delete Posts
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
